# Generated Card Assets

Store production-like generated card assets under this folder.

If the card is only a demo/example, store it under `assets/examples/cards/` instead.

## Folder Convention

Use:

- `assets/cards/generated/<pack-id>/<card-slug>/`

Example:

- `assets/cards/generated/pack-2026-04/money-pink-floyd/`

## Recommended Files Per Card

- `card-v1.png`
- `card-v2.png`
- `metadata.json`

Example-only naming lives in `assets/examples/cards/` and should use `.example` suffixes.

See docs for full process:

- `docs/ai-card-generation.md`
