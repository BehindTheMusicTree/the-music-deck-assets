# Example Card Assets

Store illustration-only card examples under this folder.

Use this path format:

- `assets/examples/cards/<example-id>/`

Example:

- `assets/examples/cards/card-example-001/`

Recommended files for examples:

- `card.example.json`
- `card.example-v1.png`
- `metadata.example.json`

This folder is for demos and documentation artifacts, not production-like generated assets.

For production-like generated assets, use:

- `assets/cards/generated/`
