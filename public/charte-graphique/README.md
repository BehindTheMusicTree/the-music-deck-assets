# Charte Graphique

Ce dossier contient la charte graphique du projet The Music Deck.

Utilisation :

- Placez ici tous les éléments liés à l'identité visuelle : logos, palettes de couleurs, typographies, exemples d'utilisation, etc.
- Documentez chaque élément pour faciliter la cohérence graphique du projet.

Structure suggérée :

- `logos/` : versions du logo
- `couleurs.md` : palette de couleurs
- `typographies.md` : polices utilisées
- `exemples/` : exemples d'application

N'hésitez pas à adapter cette structure selon les besoins du projet.
