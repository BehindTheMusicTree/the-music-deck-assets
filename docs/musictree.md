# MusicTree View

Defines the genre/subgenre completion view and related scoring intent.

## Scope

This document covers:

- MusicTree concept and UX role
- completion scoring layers
- progression links tied to tree growth

## MusicTree Concept

The `MusicTree` is a tree of genres and subgenres.

- Players try to fill the tree as much as possible with collected cards.
- It functions as a long-term collection and mastery view.

## Scoring Layers

Progress can be tracked at multiple levels:

- global completion score
- per-genre completion score
- per-subgenre completion score

The MusicTree can track quality in addition to coverage.

## Progression Links

Filling deeper branches can unlock:

- rewards
- missions
- bonus effects
